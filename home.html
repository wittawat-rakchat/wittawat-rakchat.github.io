As an AI and machine learning advocate, I would like to develop and adopt machine learning
as a business strategy in many businesses. Please have a look at my little projects below.

AI Chatbot Using Neural Networks (NNs)

This project aims to build AI chatbot using NNs. NLP techniques were applied to help the machines learn, predict, and reply to customers' messages. I experimented with a variety of activations and optimizers to find the best model for building an AI chatbot. NLTK libraries were utilised for the text pre-processing such as lemmatization.

Data Visualisation of the Company's Daily Visitors

Data visualisation is very useful and important for any business. It helps the company to understand trends and patterns in a huge amount of data and make decisions more easily. This study provides useful visualisations of the customer data, including the explanation and discussion of findings, in order to give a clear idea of the data to the company.

Neural Network Recommender System

The neural network recommender system was developed to improve the model accuracy of the model-based recommender system. I experimented with 8 architectures to find the model which gives the best RMSE and MAE. The model will predict the ratings on unwatched movies and recommend high predicted rating movies to a specific user.

Machine Learning Recommender System

This recommender system uses machine learning models to recommend movies to a specific user and predict the ratings that a user would give to movies. 8 algorithms were implemented and evaluated using RSME and MAE. I also used hyperparameter tuning and cross-validation techniques to improve the model accuracy and avoid over fitting.

Memory-based Recommender Systems

The memory-based recommender system has some advatages over the content-based recommender system because it will use other users’ information to make a recommendation to a specific user. This system uses cosine similarity to calculate the similarity of users or movies and then recommend movies which similar users have watched.

Content-based Recommender Systems

This recommender system recommends movies based on movie content or genres. It doesn’t require other users’ information and easy to build. However, there is no chance for users to discover new movies outside of their preferences becuase recommended movies will have similar content or same genres as movies which have been watched by users.

Product Category Prediction Using NLP Techinque

This study aims to predict product categories based on Amazon reviews by using machine learning algorithms and NLP technique. The models were improved using hyperparameter tuning technique on k-fold cross validation. Natural Language Toolkit (NLTK) libraries were implemented for the text pre-processing such as Tokenization, Lemmatization.

Predict the Number of Star Rating Based on Reviews

This study aims to predict the star ratings (1 to 5) according to Amazon reviews by using 3 machine learning algorithms to solve multi-classification problems from the text data. NLP techniques were applied to help the machines learn and predict the ratings from customers’ reviews. This study also implemented NLTK for the text pre-processing

Travel Insurance Purchase Prediction using 10 Algorithms

This project aims to predict whether customers will buy travel insurance using machine learning algorithms. The company wants to know whether customers want to buy a travel insurance package. Machine learning can give the solution to this classification problem and help the company to find strategies to sell insurance to customers.

Stock Marketing Analysis and Prediction Using LSTM

This project was focused on analysis of the Tesla, Apple, IBM, and Amazon stock prices since the last three years. A variety of Data visualization were provided in order to analyse the stock data such as moving average, daily return. The deep learning model using long short-term memory (LSTM) was built in order to predict the stock price of IBM.
